2. In how many ways of declare a variables?

  • var
    • var is an global and functional scope
    • var can be declared
    • var can be modified later
  • let
    • let is an block scope
    • let can be redeclared
    • let can be modified later
  • const
    • const is an block scope
    • const can't be redeclared
    • const can't be modified later

3. What is Hoisting?

Hoisting :Hoisting is a Javascript Mechanism where variables and function declarations are moved to top of there scope before the code execution
Note: Only Declarations are hoisted not the initilizations

4. What is an temperal dead zone ?

It is the area of a block where a variable is inaccessible until the moment computer initilizes it with a value.
TDZ occurs when we declare a variable of Let and Const keyword

5.What is the Difference between == & === Operator ?
